# Favicon Information

## Files Created

1. **favicon.svg** - Modern SVG favicon (primary)
   - Vector-based, scales perfectly
   - Displays a water droplet with cooling symbol
   - Purple gradient background (#667eea to #764ba2)
   - Represents liquid cooling system

2. **favicon.ico** - Fallback for older browsers

## Design Elements

The favicon represents the AI Data Center Liquid Cooling System:
- **Water Droplet**: Represents coolant liquid
- **Snowflake Symbol**: Represents cooling/temperature control
- **Purple Gradient**: Matches the application theme
- **Temperature Lines**: Represents temperature monitoring

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge): Use SVG
- Older browsers: Fall back to ICO format

## Technical Details

- SVG dimensions: 100x100 viewBox
- Primary colors: #667eea, #764ba2 (gradient)
- Icon is optimized for 16x16, 32x32, and 64x64 display sizes
